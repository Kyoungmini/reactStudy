React를 CDN(Content Distribution Network) 추가로 간단히 동작해보자

CDN은 인터넷 콘텐츠를 고속 전송하는 지리적으로 분산된 서버 집단으로 js,css,image와 같은 인터넷 콘텐츠를 로드하는 데 필요한 리소스를 신속하게 전송하기 위해 사용된다.

  • https://unpkg.com/react@17/umd/react.development.js : React객체로 Component를 제공한다.
  • https://unpkg.com/react-dom@17/umd/react-dom.development.js : ReactDOM 객체로 Component를 화면에 렌더하는 객체다
  • https://unpkg.com/babel-standalone@6/babel.min.js: babel 라이브러리로 JS 크로스브라우징과 JSX 제공한다,

React.Compoent

  • 상속 받은 객체나 함수를 Virtual DOM이 될 수 있게 하는 객체
  • render 함수를 재정의 해야 ReactDOM Compoent를 출력할 수 있다
