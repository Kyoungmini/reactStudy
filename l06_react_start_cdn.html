React js를 CDN으로 추가해서 간단히 동작해 보자(리액트 어플이 될 수 없다)

CDN으로 추가한 React 라이브러리

  • https://unpkg.com/react@17/umd/react.development.js : React 객체와 React.Component를 제공하는 라이브러리
  • https://unpkg.com/react-dom@17/umd/react-dom.development.js : ReactDOM 객체를 제공하는 라이브러리

React 객체

  • React : 라이브러리의 최상위 객체로 정적(static)으로 사용가능하다.
  • React.Component : Virtual DOM으로 생성가능한 노드개체인 Component를 제작하기 위한 타입
  • React.createElement() : HTML 노드 개체와 React Component를 호출 생성하는 함수

ReactDOM 객체

  • 컴퍼넌트를 렌더(브라우저의 출력) 제어하는 객체
  • ReactDOM.render(Component,selector dom(container)) : 컴포넌트를 특정 dom에 출력(마운트)해주는 함수
  • ReactDOM.unmountComponenAtNode(container) : 마운트된(출력) 컴포넌트를 dom(container)에서 제거하는 함수
