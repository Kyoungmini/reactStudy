바벨로 React에서 JSX 사용해 보자

JSX(Javascript Xml)

  • babel : type="text/babel" 인 스크립트로 작성을 하면 크로스브라우징(es5 or 6 동작하지 않는 브라우저에서 동작지원)된 js 코드로 컴파일할 수 있는 라이브러리
  • JSX : 바벨에서 지원하는 문법으로 자바스크립트에서 html을 xml 형식으로 작성하도록 돕는다.
  • React JSX : 리엑트에서 JSX를 사용하면 리엑트 컴퍼트를 생성한다
  • JSX compile js: const h1=React.createElement("h1",{},null);
  • {} : jsx에서 javascript를 사용할 수 있는 표현식
  • 래핑 : 한개의 컴포넌트는 렌더 시 반환되는 컴포넌트가 1개 여야한다. 때문에 공백의 부모 태그인 래핑을 jsx가 지원한다.
  • jsx는 html과 다르게 닫는 요소를 꼭 작성해야한다.
