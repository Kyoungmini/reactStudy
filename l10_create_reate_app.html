리엑트 앱 설치 및 분석

  1. 비주얼 스튜디오 code 설치
  2. node.js 웹팩(npm) 설치(최신버전)
  3. 설치 완료 확인은 터미널에서 "npm -v" 을 입력
  4. 리엑트 앱을 설치할 경로로 터미널 이동 "cd 경로"
  5. 터미널에 "npx create-reate-app 설치폴더" 입력
  6. 터미널에서 "cd 설치폴더" 입력 후 "npm start" 입력 => 실행

react-scrit

  • npm start : webpack-dev-server를 실행해서 빌드되지 않은 프로젝트(웹어플)을 실행, 실시간으로 저장 내역을 반영
  • npm run build : 프로젝트를 배포할 수 있는 상태로 빌드하는 명령(bableJS->js,코드를 압축)
