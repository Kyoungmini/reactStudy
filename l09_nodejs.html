node js 와 web pack

node js

  • NodeJS 는 구글 크롬의 자바스크립트 엔진 (V8 Engine) 에 기반해 만들어진 서버 사이드 플랫폼

NPM(Node Package Manager)

  • Node.js로 만들어진 패키지(package) 저장소
    • 모듈(module) : 소스코드의 집합 (파일단위)
    • 패키지(package) : 배포를 위해 사용되는 모듈 묶음 (폴더 단위의 집합)
    • 라이브러리(Library) : 공통으로 사용되는 기능의 집합 (모듈 or 패키지)
    • 소프트웨어 : 모듈이나 라이브러리, 페키지로 만들어내는 사용가능한 어플리케이션
    • 프레임워크(Framework) : 여러 라이브러리와 패키지로 소프트웨어를 제작할 수 있는 도구(라이브러리의 집합
  • 800,000개 이상의 코드 패키지가 존재하는 세상에서 가장 큰 소프트웨어 레지스트리
  • npm은 무료이고 오픈 소스 개발자들이 npm을 통해 소프트웨어를 공유함
  • 패키지 설치 및 버전, 호환성 관리를 할 수 있는 커맨드라인 유틸리티
  • package.json 으로 패키지의 버전과 의존성을 관리한다.

package.json

  • package name : 패키지의 이름으로 중앙 저장소에 배포할 때 version과 함께 필수 항목
  • version : 패키지의 버전이다.
  • description :프로젝트 설명
  • keywords : 프로젝트를 검색할 때 참조되는 키워드입니다
  • license : 해당 패키지의 라이선스를 넣어주면 된다.
