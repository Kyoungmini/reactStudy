NodeJs 와 NPM(Nodejs Package Manager)

Node Js

NodeJs는 구글 크롬 브라우저가 사용하는 자바스크립트 엔지(v8)에 기반해 만들어진 서버 사이드 플랫폼

NPM(Node Package Manager)

  • NodeJs로 만들어진 패키지(Package) 저장소
    • 모듈(module) : 소스코드의 집함(파일 단위)
    • 패키지(Package) : 배포를 위해 사용되는 모듈 묶음(폴더 단위 집합)
    • 라이브러리(Library) : 공통으로 사용되는 기능의 집합 (모듈 or 패키지)
    • 소프트웨어 : 모듈,라이브러리,패키지를 이용해 만들어낸 어플리케이션
    • 프레임워크(Framework) : 소프트웨어를 제작할 수 있는 도구
  • 800,000개 이상의 패키지가 존재하는 세상에서 가장 큰 소프트웨어 레지스트리
  • npm은 무료이고,오픈 소스 개발자들이 npm을 통해 소프트웨어를 공유
  • 현재 가장인기가 많은 언어인 JavaScript로된 node.js로 소프트웨어를 개발하고 있는 중이다.
  • package.json으로 패키지의 버전과 의존성을 관리한다.(name,version을 꼭 작성)
  • package-lock.json 패키지의 읜존성 트리에 대한 정보를 갖는다.(의존성 트리 생성)
